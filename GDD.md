# GDD — Колобок (прототип)

Веб-игра: тамагочи на главном экране + раннер. Тон колобка: взрослый ироничный братан («бро», «кайф», «не катит»), без сюсюканья, без мата и **без осуждения покупок пользователя** (советы нейтрально-ироничные).

**Стек:** vanilla JS (ES modules), HTML/CSS, Canvas 2D (раннер + трейл резки), `localStorage`. Без npm/React/TS.

**Источник правды:** `js/config.js`, `js/phrases.js`, `js/state.js`.

**Термины (главный экран):**

| Термин | Что в игре |
|--------|------------|
| **Режим резки продуктов** | Свайп — полезное: +6 ⭐; bad: убрать (бомба), 0 ⭐ |
| **Тап по продукту** | Полезное: съесть, +1 ⭐; bad: убрать (бомба), 0 ⭐ |
| **Кликер** | Тап по пустой сцене / колобку — очки и реакция, без еды |
| **Неполезное** | `kind: 'bad'` — красное свечение, тап/свайп **ничего не дают**, статы не меняются |

## 2. Верхняя панель

### 2.1. Ряд заголовка

- **Слева:** `?` (туториал), `⚙` (меню).
- **По центру:** капсула `#score-hub` — SVG-звезда, число (gradient), glow; pop `is-pop` при начислении.
- **Справа:** `✕` — модалка выхода (заглушка).

Формат: `toLocaleString('ru-RU')`. Подписи «ОЧКИ» нет.

### 2.2. Стат-чипы (полоски прогресса)

Четыре капсулы `.stat-chip`. Внутри — **полоска заполнения** (% стата):

| Стат | Ключ | Цвет fill | Подпись |
|------|------|-----------|---------|
| Сытость | `hunger` | `#F5A623` | СЫТОСТЬ |
| Жажда | `thirst` | `#4FB3F5` | ЖАЖДА |
| Здоровье | `health` | `#E25C5C` | ЗДОРОВЬЕ |
| Настроение | `mood` | `#B57BE0` | НАСТРОЕНИЕ |

- Отображение: эмодзи + **%** = `round(value / max * 100)` при `max` 120.
- **Критично:** value/max &lt; 15% → `stat-chip--critical`, пульс.
- **Тап по чипу:** тултип `значение / 120` на 1.5 с.

### 2.3. Меню ⚙

| Пункт | Поведение |
|-------|-----------|
| Звук, Вибрация, Правила, Поддержка | Заглушки |
| **Сбросить прогресс** | confirm → `gameState.resetProgress()` |

Сброс: статы → 120, очки/рекорды → 0, `foodInteractCount` → 0. Туториал и `tutorialCompleted` **не** сбрасываются.

## 3. Прогресс и очки

### 3.1. Сохранение (`kolobok_save`, version 4)

| Поле | Смысл |
|------|--------|
| `hunger`, `thirst`, `health`, `mood` | 0–120, старт 120 |
| `tapScore` | Очки с главного (кликер, еда, пустой тап) |
| `runScore` | Очки с раннеров |
| `bestDistance`, `bestScore` | Рекорды раннера |
| `badFoodTipDay` | Дата последнего туториала неполезной еды (`YYYY-MM-DD` или `null`) |
| `foodInteractCount` | Счётчик тапов/свайпов по летающей еде (лимит на экране) |

**На экране:** `tapScore + runScore` → `#score-total`.

### 3.2. Падение статов (дом)

- Тик: `statDecay.tickMs * homeSlowdown` = **15 с**.
- Цель: **−1%** отображаемой полоски (0–100%) за **90 мин** на каждый стат (`displayPercentPer90Min: 1`, при max 120 ≈ −1.2 пункта / 90 мин).
- Веса: hunger/thirst/health/mood = 1 (одинаковый темп).
- Дробный остаток копится в `_decayRemainder`, списание целыми пунктами.
- В раннере: тот же расчёт × `runMultiplier` (3×) + дрен по дистанции и столкновениям.
- Автосохранение каждые 5 с.

### 3.3. Зоны статов

| Зона | Диапазон |
|------|----------|
| red | 0–20 |
| yellow | 21–50 |
| green | 51–100 |
| gold | 101–120 |

## 5. Как колобок говорит

| Канал | DOM | Блокирует игру? |
|-------|-----|----------------|
| Idle-облачко | `#idle-chat-bubble` | Нет |
| Совет по еде / чеку | то же + `html.is-lecture-active` | `kolobokLecture.isActive()` — пустой тап сцены **работает** (закрывает совет) |
| Action prompt | `#action-prompt` | Нет |
| Карточка `#nutrition-tip` | отдельный UI | Сейчас **не** используется `kolobokLecture` (зарезерв) |

Файлы: `replySystem.js`, `kolobokLecture.js`, `phrases.js`, `speechPosition.js`.

### 5.1. Idle-облачко

**Показ:** `replySystem.showIdle(text, { animate, autoHide })`

| Триггер | Текст |
|---------|--------|
| Старт / смена mood | `pickPhrase(mood)` |
| Ротация 20 с | `rotateHomePhrase()` если не purchase/runner/lecture/tutorial/badFoodTip |
| Крит. стат &lt; 15% | раз в 30 с `getCriticalWarnPhrase` |
| Блок бега / чека | `getCantRunPhrase` / `getReceiptBlockedPhrase` |
| Распаковка | `unpackingFlow.say()` |
| Пустой тап сцены | `refreshPhrase(true)` после эффектов |

**Скрытие:** `idleHideMs` 4000 (или `speechHideIdleMs` 6000).

### 5.2. Советы по продуктам (`kolobokLecture`)

| Триггер | Шанс | Канал |
|---------|------|--------|
| Съел полезное (не bad) | 15% `nutritionChance` | `showIdle` + кружочки мыслей |
| После распаковки | 35% `purchaseReviewChance` | то же |

**Не показывается** после съедания bad-продукта.

**Закрытие:** тап по сцене/колобку/еде; авто-hide по таймеру; `dismiss()`.

### 5.3. Action prompt

`showActionPrompt(text, 'run')` при `isBurnRunReady`.

### 5.4. `getPurchaseReviewPhrase`

В коде есть, **UI не вызывает** — после чека срабатывает `showPurchaseReview` → один товар через `getNutritionTipPhrase`.

## 6. Тапалка и кликер (главный экран)

Отдельной мини-игры «тапалка» в билде **пока нет**. Реализован **домашний кликер** на сцене.

### 6.1. Виды тапа

| Жест | Куда | Модуль | Эффект |
|------|------|--------|--------|
| **Пустая сцена** | `.stage-hero` без еды | `homeSlice` → `performStageTap` | +1–3 ⭐, +1 mood, `is-tap-smile` |
| **Колобок** | `#kolobok` | тот же путь | То же |
| **Продукт (тап)** | `.home-spawn` | `pointerdown` | §7.3 |
| **Продукт (свайп)** | линия по сцене | `homeSlice` | §7.4 |

**Кликер:** `tapFx.js`, `CONFIG.clicker`.

### 6.2. Параметры кликера

| Параметр | Значение |
|----------|----------|
| Очки за тап | 1–3 |
| Частицы | до 56, жизнь 480 ms |
| Бонус mood | +1 |

### 6.3. Блокировки

`performStageTap` **не** при: `purchase`, `runner`, `tutorial`. При `kolobokLecture` — `dismiss()`, затем тап.

Спавн не блокируется lecture (`isSpawnBlocked` без lecture).

## 7. Летающие продукты: тап и режим резки

### 7.0. Типы и очки

| Тип | CSS | Тап / свайп | Очки | Статы |
|-----|-----|-------------|------|-------|
| **good** | `home-spawn--good` | полезное | **+1 ⭐** | буст §7.6 при +1000 tapScore |
| **neutral** | `home-spawn--neutral` | полезное | **+1 ⭐** | то же |
| **bad** | `home-spawn--bad` | бомба 💣 | **0** | **не меняются** |

Подсветка: мягкое свечение (зелёный / янтарь / красный), **без рамок**, лёгкая пульсация.

**Модули:** `homeSpawns.js`, `homeSlice.js`, `homeTapBurst.js`, `homeBadFx.js`, `badFoodTip.js`.

### 7.1. Спавн

| Параметр | Значение |
|----------|----------|
| Старт на экране | **5** (`minOnScreen`) |
| Макс. на экране | **5–14** (динамика) |
| Интервал новых | 2.8–4.5 с |
| Первый спавн | 800 ms |
| Время жизни | 8 с (+ fade 500 ms) |
| Запретная зона | центр ~28–72% × 38–92% |
| Шанс bad / neutral | 28% / 32% |
| Жажда | bias 40% на воду/чай/сок/молоко |

**Динамический лимит** (`foodInteractCount`):

- База **5**, каждые **4** взаимодействия с едой → **+1** к лимиту, потолок **14**.
- После каждого сбора: `recordFoodInteraction()` + `trySpawnToMax()`.

**Анти-наслоение:**

- Радиус продукта ~8.5%, мин. расстояние центров ~19%.
- Допустимое касание краёв ~2.5%.
- До 32 попыток позиции; нет места — **пропуск спавна** (не насильно в кучу).
- Синхронизация позиций с DOM (`syncPositionsFromDom`).

**Ассортимент — 47 позиций** (`homeFoods.list`):

- **good (19):** вода, чай, сок, молоко, яблоко, банан, брокколи, апельсин, морковь, салат, йогурт, огурец, помидор, виноград, клубника, груша, киви, авокадо, рыба.
- **neutral (11):** хлеб, яйца, сыр, кофе, курица, рис, паста, мясо, картошка, кукуруза, арахис.
- **bad (17):** пиво, пицца, шоколад, кола, чипсы, бургер, хот-дог, конфеты, мороженое, пончик, торт, печенье, фри, начос, сосиска, энергетик, леденец.

### 7.2. Туториал неполезной еды (`badFoodTip.js`)

Перед первым тапом/свайпом по bad **в новый календарный день** (и при первом запуске):

- Карточка в `#tutorial-overlay`, кнопка «Понятно».
- Текст: не ПП-перекус, **0 очков**, статы не трогаются.
- `badFoodTipDay` в save.

### 7.3. Тап — полезное

1. Burst FX → ghost-полёт к колобку.
2. `handleFoodCollect`: +1 ⭐, `kolobokEatAnim`, float.
3. 15% — совет `kolobokLecture` (не для bad).

### 7.4. Тап / свайп — bad

1. При необходимости — `badFoodTip` (подсказка).
2. Тап/свайп **убирают** продукт (бомба 💣), **0 ⭐**, статы не меняются.

### 7.5. Режим резки (свайп)

| Параметр | Полезное | Bad |
|----------|----------|-----|
| Старт жеста | ≥ 8 px | то же |
| Отличие от тапа | ≤ 14 px движения | то же |
| Очки | `round(1 × 1.65) + 5` = **+6 ⭐** | **0** |
| FX | половинки эмодзи 480 ms | бомба §7.4 |
| Трейл | canvas `.home-slice-blade-canvas` | то же |
| Вибрация | [14, 22, 16] | bombVibrate [36, 24, 32, 18] |

**Ввод:** `bindHomeSliceInput` на `stage`, `collectHitsOnSegment`, hitPadding 10 px.

**Не стартует** при тапе: `#nutrition-tip`, `#tutorial-overlay`, footer, иконки панели, menu, exit.

### 7.6. Буст статов от очков еды

Только при **положительных** очках от еды:

Каждые **1000** `tapScore` → **+1** ко всем четырём статам + float над точкой сбора.

### 7.7. Блокировка спавна

Не спавнится: purchase, runner, tutorial (кроме tutorial apple), **badFoodTip**. Свайп отключён: purchase, runner, badFoodTip.

## 8. «Распаковать покупки»

**Кнопка:** `#btn-receipt`, текст `CONFIG.ui.unpackButton`.

**Доступность:**

- `purchase.testMode === true` → **всегда** (сейчас в config включено).
- Иначе: хотя бы один стат **&lt; 20** (`receiptMinStat`).

### Флоу (`unpackingFlow.js`, 4 шага)

1. **Пакет:** confetti, burst, реплика.
2. **Чек:** демо-корзина (банан, сыр, яйца), авто 5 с.
3. **Орбита:** буст статов, тап по товару → реплика.
4. **Финал** → главная.

**Старт:** `pauseGameTimers`, стоп спавнов, `is-purchase-active`. **После:** `resumeTimers`, `resumeHomeVideo`, 35% совет по корзине, спавн товаров корзины.

**Видео колобка** идёт под оверлеем (не паузится).

Заголовок: «Чек · Пятёрочка / Перекрёсток / Чижик».

## 9. Раннер

**Старт:** `#btn-run`, все статы &gt; 0. `pauseKolobokVideo()` при старте.

### 9.1. Геймплей

| Механика | Описание |
|----------|----------|
| Управление | Тап / пробел — прыжок (до 2) |
| Дистанция | +1 м каждые N кадров |
| Скорость | База × состояние × статы |
| Статы в забеге | Дрен каждые 15 м; столкновение −5 health |
| Очки | `score.rate` × дистанция + pickups → `runScore` |

### 9.2. Босс

3 удара (`maxStrikes`) → game over. Дистанция ползёт; 5 с без столкновений — отставание.

### 9.3. Локации

Смена каждые **300 м**, 6 биомов.

### 9.4. Pickups

⭐ очки, 🍞 сытость, 💧 жажда, 💊 здоровье.

### 9.5. Результат

«Ещё раз» / «Домой» → `resumeHomeVideo()`, главная.

Файлы: `js/runner/*`, `runner-config.js`.

## 17. История правок (актуализация GDD)
